# QuickNote Backend

This is the backend of **QuickNote** – a personal notes web app built using **Node.js**, **Express**, and **MongoDB**.

---

### ✨ Features
- JWT-based user authentication
- Secure password hashing (bcrypt)
- CRUD APIs for notes
- MongoDB database with Mongoose
- Auth middleware

---

### 🧑‍💻 Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose
- bcrypt
- jsonwebtoken
- dotenv
- CORS
